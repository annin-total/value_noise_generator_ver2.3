How to use Value Noise Generator
To use the Value Noise Generator, start by adjusting parameters such as scale and octaves to define the noise pattern. Select
color palettes to enhance the visual appeal. Set persistence to control the texture details of your image. Once satisfied, generate
and save your creation. Feel free to experiment with different settings for varied results and unique visuals.

Scale Canvas Setting default / 50

Explanation
In Perlin noise, "Scale" refers to the frequency or zoom level of the noise pattern. A larger scale produces broader, smoother variations, while a smaller scale results in more detailed, granular patterns. Adjusting the scale allows designers to control the overall appearance and complexity of the generated noise.

Octaves Canvas Setting default / 4

Explanation
In Perlin noise, "Octave" refers to a layer of noise added to create more complex patterns. Each octave has a higher frequency and lower amplitude than the previous one. By combining multiple octaves, designers can achieve more natural-looking textures with varying levels of detail and roughness.
